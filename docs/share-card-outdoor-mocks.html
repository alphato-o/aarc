Outdoor run — share card mocks

Four directions for what the share card becomes when the run happened outside. All use the brand: pub-green, cream goblet logo, serif quote as the centrepiece. Same 4:5 portrait crop as the current card.

What outdoor data we have today vs. what each mock needs: the event log currently carries distance / pace / speed / heart rate samples — enough for mocks B and D right now. Mock A (route silhouette) needs a coarse GPS polyline (~100 points, fuzzed start/end for privacy) added to the run upload. Mock C (elevation) needs an altitude sample per metrics event. Both are small iOS-side additions.
